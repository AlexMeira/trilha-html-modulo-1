O que são Clients? índice

No desenvolvimento da web, "lado do cliente" refere-se a tudo em um aplicativo web que é exibido ou ocorre no cliente (dispositivo do usuário final).

Basicamente no modelo cliente-servidor, é oque o usuário final vê em sua tela e roda em seu navegador e os processos executados são conhecidos como Front-End.

O que são Servers? índice

O lado Servidor é tudo que acontece no servidor. O lado cliente comunica-se com o os web server através do HTTP. Sempre que o lado cliente interage com o site através de algum formulário ou pesquisa, é enviado uma request do navegador para o servidor que são respondidas pelo servidor. Os processos que ocorrem do lado servidor são chamados de Back-End.
